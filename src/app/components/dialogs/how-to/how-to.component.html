Mat Flip

Challenge your memory and/or math skills!

How to Play:

  1. Select a game mode
  2. Select a difficulty level
  3. Select Start Game when ready!

Game Modes:

  • Pairs: select cards to find matching pairs before time runs out!

    • View difficulty information Easy:
    • 5 pairs to match
    • 30 seconds to find all pairs
    Medium:
    • 6 pairs to match
    • 40 seconds to find all pairs
    Hard:
    • 8 pairs to match
    • 50 seconds to find all pairs
    Expert:
    • 10 pairs to match
    • 60 seconds to find all pairs
    Mastery:
    • 12 pairs to match
    • 60 seconds to find all pairs

  • Solution: memorize displayed answers shown for a brief period before they flip (5 seconds). Once they flip over, you must solve a math problem, recall the location of the correct answer, and select it before time runs out. Get as many solutions as you can before you run out of allowed fails!

    • View difficulty information Easy:
    • 4 fails allowed
    • 20 seconds to solve problems
    • Add & subtraction only
    • No negative digits
    Medium:
    • 4 fails allowed
    • 20 seconds to solve problems
    • Add & subtraction only
    • Negative digits enabled
    Hard:
    • 3 fails allowed
    • 15 seconds to solve problems
    • Add, subtraction, multiplication, division
    • Negative digits enabled
    Expert:
    • 3 fails allowed
    • 10 seconds to solve problems
    • Add, subtraction, multiplication, division
    • Negative digits enabled
